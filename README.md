# Spaceship Shooter Game
  This project meets the requirements of a basic game structure and functions. This game is created in the “Android Studio”, where and build into an Apk file. The avatars used in games are gathered from the internet and edited into the required resolutions.
  In this game application the player can control his spaceship which is placed at the bottom of the screen during the game play. The player can move in two degrees of freedom. There is only one enemy spaceship, which fires in constant time at random points. The player’s spaceship can fire consecutively as per player move and can quickly move on touching the screen. Player has three lives for scoring points by hitting the enemy spaceship in this game.
